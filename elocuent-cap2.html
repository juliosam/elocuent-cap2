Eensayo y ejercicios del capitulo 2 de Elocuent JavaScript

Este capitulo nos habla de como una expresion de javascript puede ser desde un valor como un numero o string, hasta un valor entre parentesis (extenso). Compara a las expresiones con sub-oraciones y que de la misma manera que estas, una expresion puede ser formada de otras expresiones. Y si una expresion es una sub-oracion entonces una declaracion de javascript seria una oracion completa.
